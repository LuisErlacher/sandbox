## Notas para POC

Para uma POC funcional, você pode:
- Usar serviços de placeholder como Unsplash (https://unsplash.com) ou Pexels (https://pexels.com)
- Usar geradores de placeholder como placeholder.com
- Criar imagens simples usando ferramentas de design

**Exemplo de URLs placeholder:**
- Hero: `https://images.unsplash.com/photo-1551717743-49959800b1f6?w=1200`
- Serviços: `https://via.placeholder.com/300x300?text=Servico`
- Logo: Criar um logo simples ou usar texto estilizado

## Otimização

Todas as imagens devem ser otimizadas para web:
- Comprimir JPGs (qualidade 80-85%)
- Usar formato WebP quando possível (com fallback)
- Redimensionar para tamanhos adequados
- Lazy loading implementado no HTML (atributo loading="lazy")
